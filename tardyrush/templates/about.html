About

tardyrush is written in Python using the Flask microframework. Its code is on GitHub.

tardyrush uses Twitter Bootstrap. All icons except for the copy icon are provided by Glyphicons and are included with Bootstrap. The copy icon is from famfamfam. I know all Bootstrap sites look the same, and I know I'm using the default style. I don't care.

tardyrush uses jQuery, DataTables, and ZeroClipboard.

tardyrush uses Google's CDN for jQuery, BootstrapCDN for Bootstrap, and Microsoft's CDN for DataTables.

tardyrush is powered by Steam.
